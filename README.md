# glossary

MillCloud 词汇表网页版。

同一个含义的中文文本，出现在不同项目内时，可能会被翻译成不同的英文文本，比如 `分店` 可能会出现 `branch` 和 `store` 两种翻译。这种情况导致了在不同项目间频繁流动的开发者需要花额外的时间去了解文本含义。

此外，某些专业术语难以理解或翻译，往往会使开发者困惑，比如 `vendor` 或 `chunk`。

该项目旨在 **统一** 跨项目的中英文本对照，同时也提供 **部分专业术语对照**。
